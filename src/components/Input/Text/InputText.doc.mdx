# Input.Text

The `Input.Text` component is a robust, accessible text-entry field for form and data input.  
It supports **variants** (`Outline`, `Filled`), optional **labels** (with toggleable visibility), **start and end adornments** (icons, buttons, or text), and a contextual **message area** for `error`, `info`, or `success` feedback.

The component ensures proper `aria-*` attributes (`aria-describedby`, `aria-invalid`, etc.), keyboard navigation, and ref forwarding, making it easy to integrate with forms, validation logic, and higher-order components.
