# Button

The `Button` component is a flexible, accessible, and customizable UI element designed for a wide range of interactive scenarios.
It supports multiple **sizes** (`Small`, `Medium`, `Large`) and **variants** (`Primary`, `Secondary`, `Warning`), optional **left and right icons**,
and a **loading state**. The component handles proper focus management, `aria-*` attributes, and ref forwarding for integration with forms
or higher-order components.
